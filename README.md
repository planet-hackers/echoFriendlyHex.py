# echoFriendlyHex.py

```
usage: echoFriendlyHex.py [-h] [-f FILE] [-r]

dump generally usable hex from binary and vice-versa

optional arguments:
  -h, --help            show this help message and exit
  -f FILE, --file FILE  filename (otherwise read data from stdin)
  -r, --reverse         reverse a previously dumped hex
```
